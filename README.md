# NewsExplorer

## Introduction

**NewsExplorer** is a React-based news search application that allows users to search for articles, save their favorites, and manage their personal news collection. The project demonstrates proficiency in modern web development, including React component development, API integration, user authentication, and responsive design.

### Project Objectives and Goals

- Create a responsive web application for news discovery across all devices
- Implement user authentication for personalized article saving
- Integrate with News API to fetch real-time news articles
- Build a scalable frontend architecture using React best practices

## Main Body: What Was Done and How

### Technology Stack

- **React 19.2.0** - UI library for building interactive interfaces
- **React Router DOM 7.10.1** - Client-side routing
- **Vite 7.2.4** - Build tool and dev server
- **CSS with BEM Methodology** - Maintainable styling
- **News API** - External news data source
- **Local Storage** - Token management for authentication

### Key Features

**1. News Search Functionality**

- Search form with validation
- Integration with News API (last 7 days of articles)
- Responsive grid layout displaying article cards
- "Show More" button for progressive loading
- Error handling for failed API requests

**2. User Authentication**

- User registration and login system
- Token-based authentication with localStorage
- Modal dialogs for registration and login
- Session persistence across page reloads

**3. Article Saving and Management**

- One-click bookmarking of articles
- Dedicated Saved News page
- Delete functionality for saved articles
- Visual indicators for saved articles

**4. Responsive Design**

- Mobile-first approach with breakpoints
- CSS Grid adapting from 3 columns (desktop) to 1 column (mobile)
- Touch-friendly interface
- Optimized for all screen sizes
